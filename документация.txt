edu.itstep.solarSystem
пакеты:
	conf:
		содержит классы для чтения строк из файлов(ConfReader) и преобразования их в карту(ConfProvider)
		для осущесвтления запросов

	connection:
		осуществляет connection к базе данных

	constants:
		содержит константы которые указывают путь к файлам от куда беруться запросы и данные для 
		подключения к БД

	dao:
		пакет осуществляет реализацию запросов

	dto:
		нужен для манипулирования данными
		например если у нас есть пользователь то метод findOne вернет все поля модели
		даже те кторые возвращать нельзя, такие как логин и пароль 
		и чтобы этого небыло, нужен dto

	exception:
		пакет содержитт исключения, которые могут возникнуть при работе с БД

	init:
		устанавливает начальные значения в синглетоны

	model:
		содержит модели, которые есть в БД

	service:
		осуществляет реализацию методов из пакета dao

	singleton:
		содержит синглетоны connection и запросов к БД
